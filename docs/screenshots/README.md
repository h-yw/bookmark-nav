# 截图清单

本目录用于存放 README、GitHub Release 和浏览器商店上架所需的产品截图。

## 必需截图

- `search-home.png`：新标签页首页，包含侧边栏、搜索栏和书签网格
- `bookmark-report.png`：整理报告，包含重复链接、长期未打开和失效链接检测
- `settings-drawer.png`：设置面板，包含搜索、展示和数据分组
- `batch-confirmation.png`：批量删除或移动确认弹窗

## 待确认文件

- `image copy.png`：发布前请重命名，或替换为有明确用途的截图文件名

## 商店截图注意事项

- Chrome Web Store 支持 1280x800 或 640x400 等截图尺寸。
- Edge Add-ons 接受类似横向截图，但提交前仍需确认最新尺寸要求。
- 只使用演示书签。不要包含私人 URL、内部域名、访问 token 或公司内部链接。
