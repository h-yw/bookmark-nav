# Screenshot Checklist

This directory is reserved for product screenshots used by README, GitHub Releases, and browser store listings.

## Required Shots

- `newtab-home.png`: New tab home with sidebar, search bar, and bookmark grid.
- `search-results.png`: Search result view with matched bookmarks and web search fallback.
- `bookmark-report.png`: Bookmark report with duplicate, stale, and dead-link sections.
- `settings-drawer.png`: Settings drawer with search, display, and data sections.
- `batch-confirmation.png`: Batch delete or move confirmation dialog.

## Store Notes

- Chrome Web Store supports screenshots such as 1280x800 or 640x400.
- Edge Add-ons accepts similar landscape screenshots, but final dimensions should be checked before submission.
- Use demo bookmarks only. Do not include personal URLs, private domains, access tokens, or internal company links.
