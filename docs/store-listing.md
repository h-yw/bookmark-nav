# 商店描述文案

本文档存放 Chrome Web Store 和 Microsoft Edge Add-ons 可直接使用或改写的上架文案。

## 产品名称

Bookmark Nav

## 简短描述

把浏览器书签变成可搜索、可整理的新标签页导航台。

## 一句话卖点

Bookmark Nav 用本地优先的书签导航页替换新标签页，让搜索、浏览和轻量整理书签更高效。

## Chrome Web Store 描述

Bookmark Nav 会把浏览器书签转换成一个清爽、可搜索的新标签页。

你不必每次都打开浏览器原生书签管理器。打开新标签页后，就可以按文件夹浏览书签、按标题或域名搜索、打开常用和最近书签，并查看整理建议。

主要功能：

- 新标签页书签导航台
- 基于现有浏览器书签树的文件夹侧边栏
- 本地快速搜索标题、URL、域名和文件夹路径
- 高级搜索语法：多关键词、排除词、`site:domain` 和 `@folder`
- 常用书签和最近打开视图
- 书签卡片展示 favicon、标题、域名和文件夹上下文
- 单个书签操作：复制、编辑、移动、删除
- 批量操作：复制、移动、删除已选书签
- 书签整理报告：重复链接、相似链接、空文件夹、标题异常、长期未打开和失效链接检测
- 本地设置导入/导出
- 最小权限：只使用 `bookmarks` 权限

Bookmark Nav 以本地优先为设计原则。它不需要账号，不向网页注入脚本，也不会把你的书签列表上传到服务器。

删除、批量删除、移动、清空历史、恢复默认设置和清理本地数据等高影响操作，都会使用应用内确认弹窗。

扩展会把浏览器原生书签树作为唯一真实来源。它帮助你在新标签页里搜索、打开和轻量整理书签，但不会替代浏览器完整的书签管理器。

## Microsoft Edge Add-ons 描述

Bookmark Nav 会把 Edge 书签转换成一个可搜索的新标签页导航台。

你可以用它浏览书签文件夹、按标题和域名搜索、打开常用或最近书签，并查看重复链接、长期未打开书签、疑似失效链接等整理建议。

功能亮点：

- 用书签导航台替换新标签页
- 使用现有 Edge 书签文件夹
- 搜索标题、URL、域名和文件夹路径
- 支持 `site:github.com`、`@work` 等高级搜索过滤
- 本地记录常用和最近打开书签
- 支持复制、编辑、移动、删除和批量操作
- 提供书签整理报告
- 本地保存设置、历史和整理记录
- 只使用 `bookmarks` 权限

Bookmark Nav 不向网页注入 content scripts，也不会把你的书签列表发送到远端服务。

## 权限说明

### `bookmarks`

Bookmark Nav 需要 `bookmarks` 权限用于：

- 读取浏览器书签树，并在新标签页展示
- 本地搜索现有书签
- 在界面中展示书签文件夹和路径
- 仅在用户明确确认后编辑、移动或删除书签
- 浏览器书签变化后刷新界面

扩展不申请额外 host permissions，也不声明 `content_scripts`。

## 隐私说明

Bookmark Nav 采用本地优先设计。

- 不需要账号。
- 书签数据通过浏览器 bookmarks API 读取。
- 设置、常用/最近历史、操作快照和最近一次失效链接检测记录保存在扩展本地 storage/localStorage 中。
- 导入/导出只处理用户主动选择的本地文件。
- 扩展不会把用户书签列表上传到服务器。
- 扩展不会向网页注入脚本。

展示 favicon 时，浏览器可能会请求对应网站的 `/favicon.ico`。如果加载失败，Bookmark Nav 可能会使用 DuckDuckGo favicon 服务作为图片兜底来源。

失效链接检测只会在用户手动开始检测时，对书签 URL 发起浏览器请求。检测不会在后台自动运行。

## 截图说明文案

- 新标签页导航台：在清爽、可搜索的新标签页中浏览所有书签。
- 搜索结果：按标题、域名、URL 或文件夹路径查找书签。
- 整理报告：在执行操作前查看重复、长期未打开和疑似失效链接。
- 设置：配置搜索、展示和本地数据选项。
- 批量确认：修改浏览器书签前确认高影响操作。
